# COMET Framework — Simulation Suite

This directory contains simulation scripts for the **Contextual Maturity and Organizational Readiness (CMOR) / COMET Framework**, modelling telecom operator maturity progression from NOC to SOC.

## Scripts

### `scenario_runner.py`
Generates synthetic data for three operator types (Advanced, Mid-tier, Lagging) and simulates 36-month maturity evolution using a Markov chain model.

- **Inputs**: Operator profiles with 7 maturity dimensions (scored 0–5)
- **Outputs**: COMET indicators (MCI, SGMI, AV, CRC, CMF, TS), comparative summary table, monthly MCI evolution
- **Saved artefact**: `simulation_results.npz`

```bash
python scenario_runner.py
```

### `sensitivity_analysis.py`
Performs parameter sweeps to analyse how key parameters influence COMET outputs.

- **Parameters varied**: beta (SGMI weighting), rho (AV scaling), lambda (MTTR decay)
- **Outputs**: Sensitivity tables, elasticity summary, per-dimension MCI impact
- **Saved artefact**: `sensitivity_results.npz`

```bash
python sensitivity_analysis.py
```

### `plot_results.py`
Creates publication-quality matplotlib figures (Chinese-compatible fonts).

1. **MCI evolution** over 36 months (line plot, 3 operator types)
2. **DFI vs MTTR** relationship (curve plot)
3. **Cost-Benefit function Π(T)** over time (line plot)

Figures are saved to `../paper/figures/` as PNG files.

```bash
python plot_results.py
```

> **Note**: Run `scenario_runner.py` first if you want the MCI plot to use Markov-chain data. Otherwise `plot_results.py` generates its own fallback data.

## COMET Indicators

| Indicator | Full Name | Formula |
|-----------|-----------|---------|
| MCI | Maturity Composite Index | Weighted average of 7 dimensions |
| SGMI | Service Graph Maturity Index | (1−β)·MCI + β·(DF+SG)/10 |
| AV | Availability | 1 − (1−ρ)·exp(−ρ·MCI) |
| CRC | Cognitive Readiness Coefficient | Weighted combination of AI, Culture, Org |
| CMF | Change Momentum Factor | Normalised month-over-month score delta |
| TS | Transformation Score | Weighted combination of MCI, SGMI, AV, CRC, CMF |
